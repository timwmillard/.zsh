# zsh Config

## Setup Symbolic Links

```sh
ln -s .config/zsh/zshrc ~/.zshrc
ln -s .config/zsh/zprofile ~/.zprofile
ln -s .config/zsh/oh-my-zshrc ~/.oh-my-zshrc
```
